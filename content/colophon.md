This site is built with [Hugo](https://gohugo.io/) and hosted on [GitHub Pages](https://pages.github.com/).

The homepage uses [IBM Plex Mono](https://fonts.google.com/specimen/IBM+Plex+Mono) by IBM. The rest of the site uses Verdana.

The chat widget lets you talk to an AI version of me. It runs on a [Cloudflare Worker](https://workers.cloudflare.com/) with conversation history stored in [Cloudflare D1](https://developers.cloudflare.com/d1/). The LLM backend is swappable - currently using a free-tier provider for the proof of concept.

Most of the code, including the reading list with Kindle highlights import and the chat widget, was written with the help of [Claude Code](https://claude.ai/code) by Anthropic.

The source code is available on [GitHub](https://github.com/kintecus/kintecus.github.io).
